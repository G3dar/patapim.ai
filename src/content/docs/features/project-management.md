## Overview

PATAPIM organizes work into projects within a workspace stored at `~/.patapim/workspaces.json`.

## Adding Projects

1. Click **Projects** in the sidebar
2. Click **Add Project**
3. Select a folder

## File Explorer

Sidebar file tree shows up to 5 levels deep, excluding `node_modules` and `.git`. Click a file to open in the overlay editor.

## Project Isolation

Switching projects switches all terminals, file tree, tasks, and GitHub panel. Session state is preserved per project.

## Plan Limits

- **Free**: Up to 3 active projects
- **Pro / Lifetime**: Unlimited

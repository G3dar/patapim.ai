## 概要

**xterm.js 5.3**と**node-pty 1.0**による本物の疑似ターミナル（PTY）を最大9つ同時にサポートします。

## 表示モード

### タブ表示（デフォルト）
タブで表示。ダブルクリックで名前変更。

### グリッド表示
`Ctrl+Shift+G`で切替。レイアウト：2x1、2x2、3x1、3x2、3x3。ドラッグハンドルでセルサイズ変更可能。

## ショートカット

| ショートカット | アクション |
|---------------|-----------|
| `Ctrl+Shift+T` | 新しいターミナル |
| `Ctrl+Shift+W` | ターミナルを閉じる |
| `Ctrl+1-9` | ターミナルNに切り替え |
| `Ctrl+Shift+G` | グリッド表示切替 |

## プラン制限
- **Free**: 最大9ターミナル
- **Pro / Lifetime**: 無制限

## 概要
ファイルの素早い表示と編集のためのオーバーレイエディタ。サイドバーのファイルツリーでファイルをクリックすると開きます。

## 機能
- **Prism.js**によるシンタックスハイライト
- ターミナルエリアの上にオーバーレイとして表示
- **Escape**で保存して閉じる

## 使い方
1. サイドバーのファイルツリーでファイルを探す
2. ファイル名をクリック
3. エディタがオーバーレイとして開く
4. 編集する
5. Escapeで保存して閉じる

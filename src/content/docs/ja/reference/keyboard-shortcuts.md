macOSでは`Ctrl`を`Cmd`に置き換えてください。

## Claude Code連携

| ショートカット | アクション |
|---------------|-----------|
| `Ctrl+K` | Claude Code開始 / クイックダイアログ |
| `Ctrl+I` | `/init`を実行 |
| `Ctrl+Shift+C` | `/commit`を実行 |

## ターミナル管理

| ショートカット | アクション |
|---------------|-----------|
| `Ctrl+Shift+T` | 新しいターミナルを開く |
| `Ctrl+Shift+W` | 現在のターミナルを閉じる |
| `Ctrl+Tab` | 次のターミナル |
| `Ctrl+Shift+Tab` | 前のターミナル |
| `Ctrl+1` ～ `Ctrl+9` | ターミナル1-9に切り替え |

## レイアウトと表示

| ショートカット | アクション |
|---------------|-----------|
| `Ctrl+Shift+G` | グリッド表示切替 |
| `Ctrl+Shift+S` | セッションパネル切替 |
| `Ctrl+Shift+H` | 履歴パネル切替 |

## パネルとナビゲーション

| ショートカット | アクション |
|---------------|-----------|
| `Ctrl+F` | 検索パネルを開く |
| `Ctrl+,` | 設定を開く |

## 音声入力

| ショートカット | アクション |
|---------------|-----------|
| `Ctrl+Alt`（長押し） | プッシュトゥトーク — 押しながら話し、離すと文字起こし |

## CLAUDE.mdとは

Claude Codeが会話の開始時に自動的に読み取るプロジェクト指示ファイルです。プロジェクト固有の作業方法をClaudeに伝えます。

## 生成

PATAPIMプロジェクト内のClaude Codeターミナルで`/init`を実行すると自動生成されます。その後手動で編集可能です。

## 一般的な内容

- **ビルド・実行コマンド**: `npm run build`、`npm start`など
- **アーキテクチャ概要**: プロジェクトのモジュール構造とデータフロー
- **プロジェクト固有ルール**: コーディング規約、IPC通信パターンなど
- **デバッグヒント**: 環境変数、ログファイルの場所など

## ベストプラクティス

- **簡潔に保つ**: Claude Codeはセッション開始時にファイル全体を読みます
- **「方法」にフォーカス**: ビルドコマンド、規約、注意点
- **大きな変更後に更新**: ビルドシステムや規約の変更時

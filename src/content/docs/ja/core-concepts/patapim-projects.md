## ワークスペース管理

PATAPIMは`~/.patapim/workspaces.json`に保存されるワークスペースシステムでプロジェクトを管理します。

### プロジェクトの追加

1. サイドバーの**プロジェクト**ボタンをクリック
2. **プロジェクトを追加**をクリック
3. 既存のプロジェクトフォルダを選択

Freeプランは最大3プロジェクト、ProとLifetimeは無制限です。

### プロジェクトの切り替え

サイドバーのプロジェクト名をクリックすると即座に切り替わります。切り替え時：

- すべてのターミナルが新しいプロジェクトディレクトリに切り替わります
- ファイルツリーが更新されます
- タスクパネルが新しいプロジェクトのtasks.jsonを表示
- セッション状態はプロジェクトごとに保持されます

## プロジェクトファイル

| ファイル | 目的 | 生成タイミング |
|---------|------|--------------|
| `CLAUDE.md` | Claude Code向け指示 | `/init`実行時 |
| `STRUCTURE.json` | モジュールマップとIPCチャネル | 自動/手動 |
| `PROJECT_NOTES.md` | 決定事項ログ | セッション中 |
| `tasks.json` | タスク追跡 | タスクパネル経由 |
| `QUICKSTART.md` | クイックスタートガイド | 初期化時 |

## グローバル設定

`~/.patapim/`にグローバルデータが保存されます：

| ファイル | 目的 |
|---------|------|
| `workspaces.json` | プロジェクトリストとメタデータ |
| `sessions.json` | プロジェクトごとのターミナルセッション |
| `account.json` | 認証トークン、メール、ライセンス |

## ファイルエクスプローラー

サイドバーのファイルツリーは5階層まで表示し、`node_modules`と`.git`を除外します。ファイルをクリックするとオーバーレイエディタで開きます。

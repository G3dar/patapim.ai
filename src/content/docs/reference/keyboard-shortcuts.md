# Keyboard Shortcuts

Complete reference for all keyboard shortcuts in PATAPIM.

## Claude Code Integration

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+K` | Start Claude Code | Opens Claude Code in the active terminal |
| `Ctrl+I` | Run /init | Runs the `/init` skill to set up project context |
| `Ctrl+Shift+C` | Run /commit | Runs the `/commit` skill to create a git commit |

## Terminal Management

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Shift+T` | New terminal | Creates a new terminal session |
| `Ctrl+Shift+W` | Close terminal | Closes the active terminal |
| `Ctrl+Tab` | Next terminal | Switches to the next terminal tab |
| `Ctrl+Shift+Tab` | Previous terminal | Switches to the previous terminal tab |
| `Ctrl+1-9` | Switch to terminal N | Jumps directly to terminal 1-9 |

## Layout & View

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Shift+G` | Toggle grid view | Switches between single and grid terminal layout |
| `Escape` | Close panel/editor | Closes the active editor or sidebar panel |
| `Ctrl+,` | Resize sidebar | Adjusts sidebar width |

## History & Search

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+H` | Open history file | Opens the `.patapim/history.txt` file in editor |
| `Ctrl+Shift+H` | Toggle history panel | Shows/hides the history sidebar panel |
| `Ctrl+Shift+L` | Session history | Opens the session history panel |
| `Ctrl+F` | Find in terminal | Opens find dialog for terminal content |

## Voice Dictation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Alt+M` | Start/stop dictation | Toggles voice recording on/off |
| `Alt+M` | Dictation menu | Opens dictation provider and settings menu |

## Editor

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+S` | Save file | Saves the current file in the editor |
| `Ctrl+F` | Find in editor | Opens find dialog in file editor |
| `Ctrl+H` | Find & replace | Opens find and replace in editor |
| `Escape` | Close editor | Closes the active editor view |

## Sidebar & Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+P` | Switch project | Opens project switcher |
| `Ctrl+Shift+B` | Tasks panel | Opens/closes task management panel |
| `Ctrl+Shift+E` | File explorer | Toggles the file explorer panel |

## File Tree

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Arrow Up/Down` | Navigate files | Move through files and folders |
| `Arrow Right` | Expand folder | Opens a collapsed folder |
| `Arrow Left` | Collapse folder | Closes an expanded folder |
| `Enter` | Open file | Opens the selected file in editor |

## Platform Notes

- All shortcuts listed use `Ctrl` on Windows/Linux
- On macOS, use `Cmd` instead of `Ctrl` for most shortcuts
- Some shortcuts may conflict with terminal applications running inside PATAPIM
- `Ctrl+F` and `Ctrl+H` change behavior depending on whether the terminal or editor is focused â€” in the terminal they open terminal search, in the editor they open editor find/replace

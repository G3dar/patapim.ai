On macOS, replace `Ctrl` with `Cmd` where applicable.

## Claude Code Integration

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | Start Claude Code / quick ask dialog |
| `Ctrl+I` | Run `/init` in active terminal |
| `Ctrl+Shift+C` | Run `/commit` in active terminal |

## Terminal Management

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+T` | Open new terminal |
| `Ctrl+Shift+W` | Close current terminal |
| `Ctrl+Tab` | Next terminal |
| `Ctrl+Shift+Tab` | Previous terminal |
| `Ctrl+1` through `Ctrl+9` | Switch to terminal 1-9 |

## Layout and View

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+G` | Toggle grid view |
| `Ctrl+Shift+S` | Toggle sessions panel |
| `Ctrl+Shift+H` | Toggle history panel |

## Panels and Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+F` | Open find panel |
| `Ctrl+,` | Open preferences |

## Voice Dictation

| Shortcut | Action |
|----------|--------|
| `Ctrl+Alt` (hold) | Push-to-talk — hold to speak, release to transcribe |
